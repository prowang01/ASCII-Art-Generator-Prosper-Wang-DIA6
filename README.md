# ASCII Art Generator

A simple Python-based tool that converts text into beautiful ASCII art. The project was created to showcase the use of the `pyfiglet` library and to explore the basics of creating open-source software.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Make sure you have Python installed on your system. You can download it [here](https://www.python.org/downloads/).

### Installing

A step-by-step series of instructions to set up your environment:

1. Clone the repository:
   ```bash
   git clone https://github.com/prowang01/ASCII-Art-Generator-Prosper-Wang-DIA6.git
   cd ASCII-Art-Generator-Prosper-Wang-DIA6
   ```

2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the program:
   ```bash
   python ASCII_Art_Generator.py
   ```
