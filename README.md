# ASCII Art Generator

A simple Python-based tool that converts text into beautiful ASCII art. The project was created to showcase the use of the `pyfiglet` library and to explore the basics of creating open-source software.

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have Python installed on your system. You can download it [here](https://www.python.org/downloads/).

### Installing

Follow these steps to set up your environment:

1. Clone the repository:
   ```bash
   git clone https://github.com/prowang01/ASCII-Art-Generator-Prosper-Wang-DIA6.git
   cd ASCII-Art-Generator-Prosper-Wang-DIA6
   ```

2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the program:
   ```bash
   python ASCII_Art_Generator.py
   ```

## Running the Program

To test the program, simply run the script and try different text inputs. For example:

```bash
python ASCII_Art_Generator.py
```
Input a string, and the program will generate ASCII art based on your input.

## Deployment

This project does not require deployment beyond running the script locally. However, you can package it using the `setup.py` file for sharing or installation.
